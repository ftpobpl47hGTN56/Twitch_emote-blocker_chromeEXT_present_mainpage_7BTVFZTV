1. Обзор

7BTVFZ Emote Picker — это расширение для браузера Chrome, предназначенное для удобного выбора и вставки смайликов в чат Twitch. Расширение интегрирует эмоты из популярных платформ: 7TV, BTTV (BetterTTV) и FFZ (FrankerFaceZ).

✓ Основной принцип: Мы уважаем вашу приватность и минимизируем сбор и обработку личных данных.

2. Какие данные мы собираем?

2.1 Данные, которые МЫ НЕ собираем

  • ✓ Личную информацию (имя, email, адрес, телефон)
  • ✓ IP-адреса или данные о местоположении
  • ✓ Историю браузера или файлы cookies
  • ✓ Пароли или учётные данные
  • ✓ Идентификационные данные аккаунтов
  • ✓ Аналитику использования или отслеживание действий

2.2 Данные, которые сохраняются локально

Единственные данные, которые расширение сохраняет — это ваши избранные эмоты:

  • Имена эмотов (текстовые строки, например: "Kappa", "monkaS")
  • URL-адреса изображений эмотов
  • Настройки тона кожи (для эмотов с модификаторами)
  • Состояние категорий (развёрнуты/свёрнуты разделы эмотов)

2.3 Данные при использовании API

Для отображения актуального списка эмотов расширение делает запросы к публичным API следующих сервисов:

Сервис Назначение Отправляемые данные
7TV.io Получение эмотов 7TV Имя канала Twitch (в URL)
BTTV.net Получение эмотов BetterTTV ID пользователя Twitch (в URL)
FFZ.com Получение эмотов FrankerFaceZ Имя канала Twitch (в URL)
Twitch.tv API Получение информации о канале Имя и ID канала

Важно: Эти запросы отправляют только информацию о названии/ID канала — никакая личная информация не передаётся.

3. Как мы храним данные?

3.1 Локальное хранилище (Chrome Storage)

Все ваши данные о избранных эмотах хранятся исключительно локально в вашем браузере с использованием chrome.storage.local:

  • 📍 Место хранения: локальный диск вашего компьютера
  • 🔒 Доступ: только расширение 7BTVFZ может получить доступ
  • 🚫 Передача: данные никогда не отправляются на серверы
  • ⏱️ Время жизни: хранятся до удаления расширения

3.2 Что происходит при удалении расширения?

При удалении расширения Chrome все связанные с ним данные автоматически удаляются с вашего устройства.

1. Overview

7BTVFZ Emote Picker is a Chrome browser extension designed to provide convenient selection and insertion of emotes into Twitch chat. The extension integrates emotes from popular platforms: 7TV, BTTV (BetterTTV), and FFZ (FrankerFaceZ).

✓ Core Principle: We respect your privacy and minimize the collection and processing of personal data.

2. What Data Do We Collect?

2.1 Data We Do NOT Collect

  • ✓ Personal information (name, email, address, phone)
  • ✓ IP addresses or location data
  • ✓ Browsing history or cookies
  • ✓ Passwords or login credentials
  • ✓ Account identifiers
  • ✓ Usage analytics or activity tracking

2.2 Data Stored Locally

The only data that the extension saves is your favorite emotes:

  • Emote names (text strings, e.g., "Kappa", "monkaS")
  • Emote image URLs
  • Skin tone settings (for emotes with modifiers)
  • Category states (expanded/collapsed sections)

2.3 Data Used with APIs

To display the current list of emotes, the extension makes requests to public APIs of the following services:

Service Purpose Transmitted Data
7TV.io Fetch 7TV emotes Twitch channel name (in URL)
BTTV.net Fetch BetterTTV emotes Twitch user ID (in URL)
FFZ.com Fetch FrankerFaceZ emotes Twitch channel name (in URL)
Twitch.tv API Get channel information Channel name and ID

Important: These requests only send channel name/ID — no personal information is transmitted.

3. How Do We Store Data?

3.1 Local Storage (Chrome Storage)

All your favorite emote data is stored exclusively locally in your browser using chrome.storage.local:

  • 📍 Storage Location: your computer's local disk
  • 🔒 Access: only 7BTVFZ extension can access it
  • 🚫 Transmission: data never sent to servers
  • ⏱️ Lifespan: stored until extension is uninstalled

3.2 What Happens When You Uninstall?

When you uninstall the extension, Chrome automatically deletes all associated data from your device.
